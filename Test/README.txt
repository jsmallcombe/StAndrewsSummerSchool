In order to test a new installation of cern's ROOT environment is ready for the analysis exercises at the summer school in St Andrews follow these steps:
1. Navigate to this Test directory using the command line
2. Run the following command "root Test.C"
You should now see a window with a histogram fit and a message. 
3. Type ".q" to exit ROOT.

You can also run the test script from within root rather from the command line with the root command ".x Test.C"

If you encounter trouble and are unable get help locally contact james.smallcombe@liverpool.ac.uk with the steps followed and error messages encountered.
